Iota v0.0.10a

Voxel based Mesh Slicer for Color 3D Printing

Iota is a voxel based slicer that can handle textured meshes for 3d printing. Iota can generate image slices for inkjet/powder based machines, GCode files for color FDM/FFF printing with multiple extruders and mixing extruders, and DXF files for creating stacked models with laser cutters.

  • GitHub pages containing latest binaries
  • Source code on GitHub
  • Pre-alpha testing video

Milestones:

  • v0.1: successful generation of GCode for single nozzle single color printer with settings for layer height, nozzle diameter, # of shells, # of lids, and infill density
  • v0.2: automatic generation of support for flat triangles and icicles
  • v0.3: material selection for support structures and body (dual extruder)
  • v0.4: code generation for belt printers (BlackBelt)
  • v0.5: load textured models (obj), render projected texture rim, seams
  • v0.6: sort toolpath segments by color and print in dual hotend dual color
  • v0.7: support for mixing extruders and virtual extruders and waste columns
  • v0.8: dialogs to generate new printers
  • v0.9: multiple meshes per session, move, scale, and rotate meshes
  • new infills, more color mixing, material mixing, multi-mesh color, etc. etc.
