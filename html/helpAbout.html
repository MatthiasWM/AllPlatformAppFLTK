Iota v0.3.1b

Voxel based Mesh Slicer for Color 3D Printing

Iota is a voxel based slicer that can handle textured meshes for 3d printing. Iota can generate image slices for inkjet/powder based machines, GCode files for color FDM/FFF printing with multiple extruders and mixing extruders, and DXF files for creating stacked models with laser cutters.

  • Source code on GitHub
  • Pre-alpha testing video

Milestones:

  • v0.1: successful generation of GCode for single nozzle single color printer with shells, lids, and infills
  • v0.2: settings UI for build parameters and basic printer settings
  • v0.3: automatic generation of support for flat triangles
  • v0.4: material selection for support structures and body (dual extruder)
  • v0.5: code generation for belt printers (BlackBelt)
  • v0.6: load textured models (obj), render projected texture rim, seams
  • v0.7: sort toolpath segments by color and print in dual hotend dual color
  • v0.8: support for mixing extruders and virtual extruders and waste columns
  • v0.9: dialogs to generate new printers
  • v0.10: multiple meshes per session, move, scale, and rotate meshes

  • v1.0: load colored and textured STL and OBJ files, new printers, printer settings, build settings, reliable and repeatable printouts, documentation, web page, user managment, forum, automated builds and releases, etc.
